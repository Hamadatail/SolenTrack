How Sessions Work
A Session tracks total time across multiple work periods for one project. Hit Resume when you start working, Pause when you stop — it accumulates automatically. When the project is done, hit Complete to save it as a single time entry.

What is Pomodoro?
Work in focused 25-minute blocks, then take a 5-min break. After 4 blocks, take a longer 15-min break. Proven to reduce fatigue and maintain focus. Each completed work block is auto-logged to your time entries.

🔧 Setup Guide — Google API
To connect Google Calendar, Sheets or Drive you need a free Google API key. This takes about 5 minutes and is completely free:

1. Go to console.cloud.google.com
2. Create a new project called "SolenTrack"
3. Enable: Google Calendar API, Google Sheets API, Google Drive API
4. Go to Credentials → Create → OAuth 2.0 Client ID → Web application
5. Add your GitHub Pages URL as an authorised origin: https://hamadatail.github.io
6. Copy your Client ID and paste it in Settings → Integrations
